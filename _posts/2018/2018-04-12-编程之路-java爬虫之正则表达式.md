#### Java正则表达式
java 正则表达式类库包：java.util.regex该包中包含是Pattern、Matcher这两个工具类：    

1. Pattern：编译好的带匹配的模板(如：Pattern.compile("[a-z]{2}");// 取2个小写字母)；2. Matcher：匹配目标字符串后产生的结果(如：pattern.matcher("目标字符串");)。
